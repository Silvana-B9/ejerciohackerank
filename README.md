# ejerciohackerank
Aquí se encuentra la solución de uno de los ejercicios de Hacker Rank, específicamente el de Sherlock y la bestia.
Ya que se quiere encontrar el número mas grande usando los números 5 y 3, teniendo en cuenta que lo 5 deben ser divisibles entre 3 y los 3 deben ser divisibles entre 5. 
Primero se crea una variable el cual se le asigna el valor de n y un ciclo while que se encargará de hacer cumplir esos requisitos para después imprimri el número.
